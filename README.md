<h1>📱 Note & Daily Task Flutter App</h1>

<p>
This is a simple and practical Flutter app I built to manage notes and daily tasks in one place.
The focus was on keeping the code clean, readable, and easy to scale while using Provider for state management.
The app runs on both Android and iOS and supports light and dark themes.
</p>

<h2>✨ Features</h2>
<ul>
  <li>Create, edit, and delete notes</li>
  <li>Add and manage daily tasks</li>
  <li>Light & dark mode</li>
  <li>Clean and maintainable structure</li>
  <li>Cross-platform support</li>
  <li>State management with Provider</li>
</ul>

<h2>🛠 Tech Stack</h2>
<ul>
  <li>Flutter</li>
  <li>Dart</li>
  <li>Provider</li>
</ul>
